What is a Series?

A Series is a 1D labeled array — like a single column in a spreadsheet with an index.

python
import pandas as pd # Create Series sales = pd.Series([45000,52000,48000,61000], index=["Q1","Q2","Q3","Q4"], name="Sales_2024") print(sales) print("\nDtype:", sales.dtype) print("Index:", sales.index.tolist())
Output
Q1 45000 Q2 52000 Q3 48000 Q4 61000 Name: Sales_2024, dtype: int64 Dtype: int64 Index: ['Q1', 'Q2', 'Q3', 'Q4']

Series Operations

python
s = pd.Series({"Cairo":450000,"Alex":280000,"Giza":195000,"Aswan":90000}) print(s["Cairo"]) # 450000 print(s[s > 200000]) # Filter print(s.sort_values()) # Ascending print(s.sort_values(ascending=False).head(2)) # Top 2 # Stats print(f"Total: {s.sum():,}") print(f"Mean: {s.mean():,.0f}") print(f"Max: {s.max():,} ({s.idxmax()})")
Output
450000 Cairo 450000 Alex 280000 dtype: int64 Aswan 90000 Giza 195000 Alex 280000 Cairo 450000 dtype: int64 Cairo 450000 Alex 280000 dtype: int64 Total: 1,015,000 Mean: 253,750 Max: 450,000 (Cairo)

loc and iloc

python
# loc — label-based (use column NAMES) df.loc[0] # Row with index 0 df.loc[0, "Product"] # Specific cell df.loc[0:5, ["Product","Sales"]] # Range + columns # iloc — position-based (use NUMBERS) df.iloc[0] # First row df.iloc[0, 2] # Row 0, column 2 df.iloc[:5, :3] # First 5 rows, first 3 cols df.iloc[[0,2,4], :] # Specific rows # query() — SQL-like syntax result = df.query("Sales > 50000 and Region == 'Cairo'") print(result.head())

Creating Date Columns

python
import pandas as pd df = pd.DataFrame({ "Order_Date": ["2024-01-15","2024-02-28","2024-03-10","2024-04-01"], "Amount": [5000, 8500, 3200, 12000], }) # Convert string to datetime df["Order_Date"] = pd.to_datetime(df["Order_Date"]) # Extract date parts df["Year"] = df["Order_Date"].dt.year df["Month"] = df["Order_Date"].dt.month df["Day"] = df["Order_Date"].dt.day df["Quarter"] = df["Order_Date"].dt.quarter df["Weekday"] = df["Order_Date"].dt.day_name() df["Week"] = df["Order_Date"].dt.isocalendar().week print(df[["Order_Date","Month","Quarter","Weekday"]])

Order_Date Month Quarter Weekday 0 2024-01-15 1 1 Monday 1 2024-02-28 2 1 Wednesday 2 2024-03-10 3 1 Sunday 3 2024-04-01 4 2 Monday
