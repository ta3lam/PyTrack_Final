01

Python Foundations

Finish Units 1-3 first. The goal is comfort with notebooks, scripts, functions, data structures, errors, and reusable code.

  • Outcome: write clean Python scripts and notebooks.
  • Practice: solve every lesson task without opening the solution first.
02

Analysis Core

Use Units 4-6 to build the working analyst toolkit: arrays, DataFrames, missing values, outliers, types, and cleaning rules.

  • Outcome: clean a raw CSV into an analysis-ready dataset.
  • Practice: document every cleaning decision.
03

EDA and Statistics

Turn summaries into real questions: distributions, segments, correlations, confidence intervals, and hypothesis testing.

  • Outcome: produce 5 decision-ready findings.
  • Practice: use the training notebook statistics sections.
04

SQL and Data Sources

Expand beyond files: relational joins, CTEs, window functions, APIs, JSON normalization, and responsible scraping.

  • Outcome: combine database, API, and CSV data in one project.
  • Practice: rebuild a pandas analysis from SQL queries.
05

Visualization and Storytelling

Move from charts to communication: chart choice, annotation, executive summaries, accessibility, and narrative order.

  • Outcome: one-page insight report with annotated visuals.
  • Practice: rewrite chart titles as business conclusions.
06

Dashboards and Portfolio

Package the work: Plotly or Streamlit dashboards, demo apps, polished project summaries, and deployment notes.

  • Outcome: one public portfolio project.
  • Practice: ship a dashboard and a clean README.
07

ML for Analysts

Keep ML practical: train/test split, leakage, baseline models, metrics, feature importance, and business interpretation.

  • Outcome: explain model performance without overclaiming.
  • Practice: compare a baseline with one improved model.
08

Capstone Delivery

Finish with real datasets, a clean notebook, a dashboard or report, a README, and 5-7 decision-focused insights.

  • Outcome: interview-ready project evidence.
  • Practice: review the project as if it belongs to a client.
